　　　　7月7日　　　　　　　雨の七夕
　今朝方凄い雷鳴で目覚める、プチが外で寝ている筈飛び起きて薄暗いガラス戸を開けた、にゃ〜ん　直グ横のクーラーの室外機の上でないた。
　朝シャワーを試みている、血流が良くなるのか結果は良いようだ。冬はどうか判らないが当分実行したい。朝食が遅くブランチになる。先日10歳
　違いの姪が同じ事を言っていた。自分流がどんどん幅を利かせる年まで来てしまった。健康の話をテーマに店舗を借りて３ヶ月位の講習をやって
　いるグループがある。食パン３斤１００円とチラシがポストに入っていたのが１ヵ月半前、今は酵素の話だそうでご近所の方が私に勧めてくださった。
　イベントの時は体が弱い人・老人達が２・３０人話を聴いているようだ。最後に高額のサプリメントなどを買うようになってお終いになる事を何回も耳
　にしているので私は乗らないけれどお金持ちの方が夫婦で聞きに行っている。一種の信仰みたいなもので信じれば効くかもしれないからやたらな
　事は言えない。午後、懐かしい太陽のお出ましでホッとしたのも束の間、18時には土砂降りです。でもこの間に医者に行きお薬を頂いて来ました。

　そうだ。しかし肝心の箇所の字が小さくて天眼鏡が必要・致し方なし！楽しむ事を考えよう。今朝も雨だったが正午・窓に光が射してきた。明るい！！

　６月２９日　　　　　　シナモンキャンディ
　　　２９日は恵理ちゃんの誕生日！　貴女は此処へよく来てくれるそうですね・・・・・・　♪　おめでとう　♪
午前中はお天気が良かった、湿度は高かったが洗濯物は片付き、湿ったプチ用の敷物の数々を干す事が出来た。　
１３日に胃カメラをのんだ後どうも胸の辺りが本当でない、２年前の検査の後もそうだった事が記憶にあるので考える、
麻酔をかけて検査をすると患者は意識が無いので楽だし、医者は隅々まで完全に診る事が出来るから良い手段だと思っ
ていたがあの胃カメラの管は硬いから、噴門の括約筋が弛緩したのだろう、２回繰り返してようよう気がついた。私は高齢
だから弾力が無くなっていて元に戻りにくいに違いない。逆流性食道炎で食道が、ただれているのかも知れない、胃下垂
だからと、食事をして横になる習慣が裏目に出る。休息時の姿勢を考えることだ、食道に負担が掛からないように食事を良
く噛むこと、沢山いっぺんに食べない事、など　自分でなければ解らない体の訴えをキャッチして考える。
普通は医者に不調を訴え薬が出ますが私の場合、制酸剤が出る位で後は自分の養生より術は無いと解っています、幸い
以前貰った薬が有るので寝る前に飲む位で様子を見ています。　じめじめ梅雨は心身ともにマイナスに繋がリます。
梅雨が終わるのが待ち遠しいですね。
